Post Information

Insert image

Paste an image URL. The alt text will automatically be renaed to the filename, but you can change it to your own value.

Leave empty to link to the image file itself.

Insert table

Choose the number of columns and body rows. A header row is added automatically above those rows.

Example: 3 body rows creates 1 header row + 3 editable rows.

Draw.io diagram

Paste the diagrams.net / draw.io iframe embed. The editor will automatically wrap it in a white background for light and dark mode.

On export this is saved as raw HTML with a drawio-white-background wrapper.

YouTube video

Paste a YouTube URL. The editor will convert it to the Docsy-compatible iframe embed on export.

Supports normal YouTube links, youtu.be links, shorts, live URLs and embed URLs.

Docsy button

Set the button text and link. You can also edit an existing Docsy button later.

GitHub Docsy button

The text and style are fixed. Paste only the GitHub link.

This inserts: View on my GitHub page

Quiz shortcode

Create and edit the full Docsy quiz shortcode without manually writing JSON. Add questions, answers, feedback messages and section references. The intro text is added automatically.

Pasted image files use jv-media-{postId}-{sha256}.ext convention. Blob settings are stored in this browser until "Delete" button is used or cache is deleted. Draw.io diagram

      
Paste the diagrams.net / draw.io iframe embed. The editor will automatically wrap it in a white background for light and dark mode.

      

      
On export this is saved as raw HTML with a drawio-white-background wrapper.

      

    

  


  

    

      
YouTube video

      
Paste a YouTube URL. The editor will convert it to the Docsy-compatible iframe embed on export.

      

      
Supports normal YouTube links, youtu.be links, shorts, live URLs and embed URLs.

      

    

  


  

    

      
Iframe

      
Paste a URL. The editor will insert a standard iframe with a fixed 960 × 540 size.

      

      

      
Output format: <iframe frameborder="0" width="960" height="540" src="..." title="..."></iframe>
